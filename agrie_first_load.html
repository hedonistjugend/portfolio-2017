Importing stuff to Aggregion

Content Owners with huge libraries have a big need to add their content to Aggregion. Earlier it was done with the command line, but now we decided to make it a bit friendlier. So we developed a desktop app. Our app takes Content Owner's file bundles it, protects it & uploads everything to our servers.
